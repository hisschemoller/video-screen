## Raspberry Pi Zero

Manual start:

* Connect to power adapter.
* Connect RPi Mini HDMI to controller HDMI.
* `ssh pi@raspberrypi.local`
* Password: raspberry
* `/etc/init.d/videoloop start`

* On controller choose HDMI as input

No signal from RPi.

Try Thinkpad HDMI out. No, has no HDMI out.
